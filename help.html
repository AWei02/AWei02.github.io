        # 帮助手册(V1.3.4)

        顾名思义，直流阻抗(DCR)计算工具可以帮助您从HPPC设备数据中快速计算DCR，其软件界面如下：

        # 1.参数定义

        # 1.1 必填项

        根据实际情况输入各项参数。其中，红框为必填项，其余为生成excel辅助标识(可不填)。

        # 1.2 文件浏览

        点击浏览按钮即可弹出文件选择框，可选择多个平行样同时计算。填入sheet名以确定要检索的范围。

        注意：当需要查找多个名称相近的sheet表时（如详细数据1、详细数据2、详细数据3），可输入相同的首段部分（如：详细数据），程序会自动拼接符合要求的sheet表。

        # 1.3 列名定义

        计算DCR需要测试数据中一些必要的表头名称，下图为HPPC测试实例(红框为DCR需要的信息)。

        下图①对应为上述的时间、电压、电流、循环序号(若存在)

        下图②对应上述的工步号和工步名称(若需要变更则点击③)

        # 1.4 时间节点

        红框内提供6个时间点可输入数值(支持小数)即可，请注意时间单位是秒。

        该工具目前仅针对(h: m: s. ms)时间格式做了识别，未来遇到新的时间需要兼容时，请联系应用仿真开发部。

        # 2. 开始计算

        参数填写完成后，点击开始计算DCR

        可在日志中查看任务执行情况

        ①用于核对输入信息是否准确无误

        ②展示任务进程情况

        # 3. 查看和导出结果

        当日志出现①时，可点击②生成excel文件，也可点击③查看输出结果或过程量

        注意：结果导出后，如需再次使用，需要重启该工具。

        # 4.菜单操作

        为了避免繁琐的输入参数，该工具会将参数存储至tmp.txt文件中，以便于下次打开时仍保留上次操作。

        点击文件菜单，可对常用的定义参数进行保存，便于数据迭代时输入操作。

        1. 打开参数文件：载入文件，要求txt格式

        2. 保存参数文件：将当前输入的记录进tmp.txt文件中

        3. 参数文件另存为：将当前输入的另存到任意位置
